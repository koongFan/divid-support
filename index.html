자주 묻는 질문 (FAQ)

Q. Divid는 어떤 앱인가요?
Divid는 일상, 경계, 업무 세 가지 모드로 관계를 구분하여 소통할 수 있는 메신저입니다. 각 모드마다 다른 프로필을 설정할 수 있어, 상황에 맞는 모습으로 대화할 수 있습니다.
Q. 모드는 어떻게 전환하나요?
화면 상단 또는 하단의 친구 탭이나 채팅 탭을 빠르게 2번 클릭하면 모드가 전환됩니다.
Q. 같은 친구를 여러 모드에 추가할 수 있나요?
아니요, 같은 친구는 하나의 모드에만 추가할 수 있습니다. 이는 각 모드의 관계를 명확하게 구분하기 위함입니다.
Q. 친구가 제 모든 프로필을 볼 수 있나요?
아니요. 내가 상대방을 친구로 추가한 모드의 프로필만 볼 수 있습니다. 예를 들어, 내가 상대방을 일상 모드에 추가했다면 상대방은 내 일상 프로필만 볼 수 있습니다.
Q. 친구는 어떻게 추가하나요?
두 가지 방법이 있습니다:
  • 태그 검색: 친구 탭에서 검색 아이콘을 눌러 친구의 태그를 검색하여 추가 (@ 없이 태그명만 입력)
  • 연락처 동기화: 연락처 권한을 허용하면 이미 Divid를 사용 중인 연락처를 자동으로 찾아줍니다 (선택사항)
Q. 연락처 동기화를 꼭 해야 하나요?
아니요, 필수가 아닙니다. 태그 검색만으로도 친구를 추가할 수 있습니다. 연락처 동기화는 편의 기능이며, 거부해도 모든 서비스를 이용할 수 있습니다.
Q. 알림을 끄고 싶어요.
설정 > 알림 설정에서 전체 알림을 끄거나, 친구별/채팅방별로 개별 설정할 수 있습니다. 특정 친구의 알림만 끄고 싶다면 채팅방에서 알림 설정을 변경하세요.
Q. 차단 기능은 어떻게 사용하나요?
친구 프로필 화면에서 차단 버튼을 누르면 됩니다. 차단한 사용자는 더 이상 나에게 메시지를 보낼 수 없으며, 설정 > 개인정보 보호 > 차단 관리에서 차단 목록을 확인하고 해제할 수 있습니다.
Q. 프로필 사진은 어떻게 변경하나요?
각 모드의 친구 탭에서 상단의 내 프로필을 클릭하면 프로필 편집 화면으로 이동합니다. 여기서 이름, 상태 메시지, 프로필 사진을 변경할 수 있습니다.

문제 해결

로그인이 안 돼요

  • 소셜 로그인(Apple, Google, Kakao) 계정을 확인해주세요
  • 네트워크 연결을 확인해주세요
  • 앱을 최신 버전으로 업데이트해주세요
  • 계정을 삭제한 경우 30일 유예기간 동안 재로그인이 불가능합니다

푸시 알림이 안 와요

  • 설정 > 알림 설정에서 알림이 켜져 있는지 확인
  • iOS 설정 > 알림 > Divid에서 알림 권한 확인
  • 방해금지 모드가 켜져 있지 않은지 확인
  • 특정 친구의 알림을 끄지 않았는지 확인

친구 추가가 안 돼요

  • 태그를 정확하게 입력했는지 확인 (@ 없이 태그명만 입력)
  • 이미 다른 모드에 친구로 추가된 사용자가 아닌지 확인
  • 네트워크 연결 상태 확인

채팅이 느려요

  • 네트워크 연결 상태 확인 (Wi-Fi 또는 LTE/5G)
  • 앱을 완전히 종료 후 재시작
  • 최신 버전으로 업데이트

사진을 업로드할 수 없어요

  • iOS 설정 > Divid > 사진 접근 권한 확인
  • 저장 공간이 충분한지 확인
  • 사진 파일 크기 확인 (최대 10MB)
